Opening keychain Files

There are three different ways that you can open keychain files for decoding. Note that when KeyChainDD starts, it doesn't display a blank window - a KeyChainDD window will open only one you open a file.

Drag and drop
Simply start KeyChainDD, and then drag and drop the file that you want to decode onto the KeyChainDD icon in the Dock. For example, if you receive password file as part of an e-mail, just drag the file from your e-mail application and drop it the KeyChainDD icon.
Open from the File menu
You can open a keychain file from the "Open..." menu item under KeyChainDD's File menu.
Cntrl-Click or right click
KeyChainDD registers itself as able to open .DAT files when you first run it. Once the registration process in complete, you can open a .DAT file by right-clicking, or cntrl-clicking the file, then selecting either "Open" or "Open With".

Once KeyChainDD has opened the file, it will display a list of attachments in the keychain file. Note that not all keychain files actually contain attachments - some contain only e-mail attributes such as font and formatting information. In such cases, the window will not contain any files.
