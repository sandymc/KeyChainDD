Drag-and-Drop operations

Drag and Drop operations are easiest way to use KeyChainDD. Here's how:

The steps to drag and drop with KeyChainDD are:

Display the keychain that contains the item you want use.

Use KeyChainDD's main window to navigate to the item you want. KeyChainDD displays items in an outline view; you can click on the "expand icon" (the little triangle next to each keychain) to show what's inside the keychain.

Unlock the keychain.

You may need to unlock the keychain. If the keychain is locked, the text describing items in it will be greyed out, and the padlock icons next to the name of the keychain will show as closed. To unlock the keychain, either double-click on the keychain item that you will be using, or click the key icon in the lower left of the KeyChainDD window. A dialog will display asking for your password to the keychain. Note that this dialog is not a KeyChainDD dialog, but is from the OS X security manager. KeyChainDD never sees the password that you provide. Note: If this is the first time that you are using KeyChainDD, or you have upgraded KeyChainDD, the OS X security manager may also ask for your permission to allow KeyChainDD to access the items in the keychain.

Drag the password and user id information.

To drag the information, click, hold and drag on either the "User Id" or "Password" buttons. Note that unless the selected item is unlocked, they will be greyed out, and you will not be able to drag without entering a keychain password.

Once you have performed the drag operation, KeyChainDD will automatically erase any traces of the information that you dragged still held in the OS X drag manager.
