Using KeyChainDD as a service

The most secure way to use KeyChainDD is from the services menu of the application you want to paste to. You can use KeyChainDD as a service either when it is open (running actively) or when it is closed, but its mode of operation is slightly different in each case:

The steps to use KeyChainDD as a service when KeyChainDD is open are:

Display the keychain that contains the item you want use.

Use KeyChainDD's main window to navigate to the item you want. KeyChainDD displays items in an outline view; you can click on the "expand icon" (the little triangle next to each keychain) to show what's inside the keychain.

Unlock the keychain.

You may need to unlock the keychain. If the keychain is locked, the text describing items in it will be greyed out, and the padlock icons next to the name of the keychain will show as closed. To unlock the keychain, either double-click on the keychain item that you will be using, or click the key icon in the lower left of the KeyChainDD window. A dialog will display asking for your password to the keychain. Note that this dialog is not a KeyChainDD dialog, but is from the OS X security manager. KeyChainDD never sees the password that you provide. Note: If this is the first time that you are using KeyChainDD, or you have upgraded KeyChainDD, the OS X security manager may also ask for your permission to allow KeyChainDD to access the items in the keychain.

Paste the user id and password information via the services menu.

In the application into which you want to paste, first place the text cursor in the location that you want to paste to, then open the services menu. This is usually under the first menu, the one with the name of the application - e.g., "Safari" for the default browser. In this menu, select the KeyChainDD item, and then either the "Password" or "User Id" sub-menu items. KeyChainDD will then paste the item that you selected. Note: if KeyChainDD is open, you must first selected the item you want to paste in KeyChainDD. If it is already open, KeyChainDD will not prompt for an item, but will use the current item.

The steps to use KeyChainDD as a service when KeyChainDD is closed (not running) are:

Paste the user id and password information via the services menu.

If KeyChainDD is closed when you select it from the services menu, KeyChainDD will automatically start, and display a dialog that will allow you select a item from which to obtain a user id or password. Once the dialog is displayed, select the item that you want, and click ok. If necessary, you will be prompted for a keychain password.

Once you have performed the services operation, KeyChainDD will automatically erase any traces of the information still held in the OS X drag manager.
